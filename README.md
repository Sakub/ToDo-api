# API for ToDo

### Requirements
- [node.js]

### Setup
- Clone this repository: `git clone https::github.com/Sakub/toDo-api.git`
- Go to the repository: `cd toDo-api`
- Install needed packages: `npm install`
- Run a project: `npm run start`
